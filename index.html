Quais são os órgãos para regulamentar a internet no Brasil?


Lei Carolina Dieckmann (Brasil - 2012)

A Lei Carolina Dieckmann, sancionada em 2012 no Brasil, tipifica como crime a invasão de computadores, dispositivos móveis e outros equipamentos eletrônicos para obtenção de dados sem autorização. A lei também prevê penas mais severas para os crimes de furto e roubo de senhas e dados pessoais. A legislação foi criada em resposta ao caso da atriz Carolina Dieckmann, que teve fotos íntimas divulgadas na Internet após ter seu computador hackeado.


Marco Civil da Internet (Brasil - 2014)

A Lei Marco Civil da Internet é uma lei aprovada em 2014 no Brasil que estabelece regras para o uso da Internet no país,- como a neutralidade da rede, a proteção de dados dos usuários e a responsabilização de provedores de aplicação.


Lei Geral de Proteção de Dados - LGPD (Brasil - 2018)

A Lei Geral de Proteção de Dados (LGPD) é uma lei brasileira de 2018 que estabelece regras para a coleta, armazenamento e uso de dados pessoais pelas empresas, exigindo consentimento dos titulares, transparência, segurança e direitos aos titulares dos dados. A LGPD prevê sanções em caso de descumprimento.

Quais são os principais aspectos de validação para páginas acessíveis?

Para validar a acessibilidade de uma página web, é importante verificar aspectos como a marcação HTML, o contraste das cores, a navegação por teclado, a legibilidade do conteúdo, o uso de alternativas de texto para imagens, a compatibilidade com leitores de tela e a funcionalidade do formulário. A acessibilidade deve ser uma preocupação contínua para garantir que todas as pessoas possam acessar e usar as informações e funcionalidades da página de forma independente e igualitária.

Como criar projetos acessíveis?

Para criar projetos plenamente acessíveis, é importante pesquisar as necessidades das pessoas com deficiências,
usar padrões de acessibilidade, considerar a usabilidade e testar com usuários com diferentes tipos de deficiências.
